# g_map

صفحة هبوط عربية لخدمة إضافة النشاط التجاري على جوجل ماب، بتصميم حديث ومتجاوب.

## نظرة سريعة

المشروع يحتوي على صفحتين HTML جاهزتين:

- صفحة هوية رابط لتقنية المعلومات: map.html
- صفحة بديلة بتصميم مختلف: index.html

كما يحتوي على ملفات صور داخل مجلد img.

## المميزات الحالية

- تصميم حديث ومتجاوب للجوال والتابلت وسطح المكتب
- خطوط عربية محسنة
- أيقونات Font Awesome
- زر واتساب عائم
- زر العودة لأعلى مع ظهور تدريجي أثناء التمرير
- قسم باقات (3 باقات) مع روابط واتساب ورسائل جاهزة
- روابط السوشال ميديا مضافة
- favicon مضاف للموقع

## هيكل المشروع

.
├─ map.html
├─ index.html
└─ img/
├─ rit.webp
└─ favicon.webp

## طريقة التشغيل

1. افتح المشروع في المتصفح مباشرة عبر أحد الملفين:

- map.html
- index.html

2. أو افتح المجلد في VS Code واستخدم Live Server لعرض الصفحة بشكل حي.

## التخصيص السريع

- رقم التواصل والواتساب:
  ابحث عن الرقم 966533116385 واستبدله برقمك.

- روابط السوشال ميديا:
  عدل روابط LinkedIn / Instagram / X / Facebook داخل الفوتر.

- الباقات والأسعار:
  عدل أسماء الباقات والمزايا والأسعار داخل قسم الباقات في map.html.

- الهوية البصرية:
  عدل ألوان :root داخل CSS لتغيير الطابع العام بسهولة.
